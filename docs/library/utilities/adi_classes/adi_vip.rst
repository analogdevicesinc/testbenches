VIP
================================================================================

Overview
-------------------------------------------------------------------------------

VIP related base classes.

adi_agent
-------------------------------------------------------------------------------

Extends the adi_component base class. It is the base class for agent classes.
For the partent, only adi_environments are valid, as it doesn't make sense for
other classes to contain an adi_agent class.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

No additional variables are available for this class.
