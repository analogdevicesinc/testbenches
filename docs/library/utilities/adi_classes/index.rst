ADI class hierarchy
================================================================================

.. toctree::
   :hidden:

   Common <adi_common>

The ADI class hierarchy is split into a couple of files that hold the relevant
class declarations.

Contents
-------------------------------------------------------------------------------

#. :ref:`adi_common`: Base classes that are common with all ADI classes
#. :ref:`adi_datatypes`: Base classes for datatypes (FIFO and LIFO)
#. :ref:`adi_environment`: Environment related base class
#. :ref:`adi_vip`: VIP related base classes (agent, driver, monitor and sequencer)
#. :ref:`adi_api`: API related base classes (API, regmap, register and field)
